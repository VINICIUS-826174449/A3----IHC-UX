# 🎓 ULIFE — Redesign do Portal Acadêmico
Trabalho A3 — Interação Humano Computador e UX/UI

O Ulife é um redesign completo do portal acadêmico da USJT, desenvolvido como trabalho A3 da disciplina de Interação Humano Computador e UX/UI. O objetivo foi identificar falhas de usabilidade no sistema original e propor uma interface corrigida e aprimorada, aplicando na prática os conceitos e heurísticas estudados ao longo do semestre.

O projeto foi construído integralmente com HTML, CSS e JavaScript puro, sem frameworks ou bibliotecas externas, e cobre as principais funcionalidades do portal: página inicial, acadêmico, financeiro, quadro de aulas e notas por disciplina.

# 📁 Estrutura de Arquivos
```ulife/
├── index.html
├── academico.html

│   ├── financeiro.css     # Estilos da página Financeiro
│   ├── qda.css            # Estilos do Quadro de Aulas
│   └── notas.css          # Estilos da página Notas
│
└── image/
    └── ulifeicon.png      # Ícone do portal
```
